projet-synthese
===============
Le projet présenté ici entre dans le cadre du projet de synthèse de seconde année de DUT
Informatique à l’IUT de Bayonne. Ce projet consiste en la réalisation d’une application permettant d’effectuer une visite virtuelle de monuments historiques et archéologiques. C’est notre tuteur monsieur Richard CHBEIR, membre du corps enseignant de l’IUT Informatique de Bayonne.
Avec l’augmentation du nombre des terminaux numériques( par exemple appareils photo, téléphones portables, …) et leurs capacités avancées de prendre des photos /vidéos, partager et publier des photos en ligne est devenu une des pratiques les plus populaires des utilisateurs du Web. Dès lors il est possible d’envisager un nouveau service qui utilise ces photos leurs métadonnées, et leurs annotations afin de virtualiser une visite d’un site touristique-archéologique.

• Le but du projet :

Permettre à tous les utilisateurs de cette application de visiter n’importe qu’elle monument ou sites historiques-archéologique.

• Les objectifs majeurs :

-Maitriser Java, l’IDE Eclipse, les différentes APIs permettant de récupérer des photos sur internet.
-Explorer les attributs EXIF des photos et de sélectionner les plus importantes pour accomplir le but de ce projet.
- Concevoir un algorithme permettant de trier et ordonner les photos.

• Les objectifs Personnels :

-Apprendre a se servir de Git
